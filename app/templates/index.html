About funicular

funicular converts transit agencies' open GTFS data into planner-friendly GIS shapefiles. Choose an agency below and funicular will prepare a zip file with the routes and stops for with the selected GFTS file contains data. funicular also calculates some useful metadata for each geographic element so you can see:

  • Routes served by each stop
  • Number of trips run on each route
  • Weekday versus weekend service
  • Service hours and trip durations
  • with more coming!

Mention GTFS Exchange

funicular is a work in progress and we invite your feedback on its usefulness, the features you like or don't like in the output, other things you'd like to see. Please send comments to us at transitshpgenerator@gmail.com.
